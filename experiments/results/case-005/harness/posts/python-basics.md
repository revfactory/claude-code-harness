# Python 기초 문법

Python은 읽기 쉬운 프로그래밍 언어입니다.

## 변수와 타입

```python
name = "Python"
version = 3.12
is_awesome = True
```

## 리스트와 딕셔너리

```python
fruits = ["apple", "banana", "cherry"]
person = {"name": "Alice", "age": 30}
```

## 함수 정의

```python
def greet(name):
    return f"Hello, {name}!"

print(greet("World"))
```

## 클래스

```python
class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        return f"{self.name} makes a sound"
```

Python은 **초보자**에게도 *훌륭한* 선택입니다.
