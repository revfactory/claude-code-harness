# Fix Bug

비동기 버그를 체계적으로 분석하고 수정합니다.

## Arguments
$ARGUMENTS - 대상 파일 경로

## Instructions

1. **분석 단계**: 대상 파일을 읽고 async-debugging 스킬에 따라 버그를 분석하세요.
2. **문서화**: bug-analysis.md에 발견된 버그 목록, 원인, 영향을 기록하세요.
3. **수정**: 각 버그를 수정하되, 헬퍼 메서드 추출과 try/finally 패턴을 적용하세요.
4. **테스트**: 동시성 시나리오를 포함한 포괄적 테스트를 작성하세요.
5. **검증**: 테스트를 실행하여 모든 버그가 수정되었는지 확인하세요.
