# 코드 리팩토링 프로젝트

스파게티 코드를 체계적으로 리팩토링하는 프로젝트입니다.

## 리팩토링 원칙
- 단일 책임 원칙 (SRP): 함수당 하나의 책임
- DRY: 중복 제거
- 매직 넘버 → 명명된 상수
- 조건 분기 → 전략 패턴
- 원본과 동일한 입출력 보장 (테스트 필수)

## 파일 구조
- constants.js - 상수 정의
- pricingStrategies.js - 전략 패턴 클래스
- calculator.js - 메인 계산 로직 (순수 함수)
- original.js - 원본 코드 (비교용)
- calculator.test.js - 동일성 + 단위 테스트
