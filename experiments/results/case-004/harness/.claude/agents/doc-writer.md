# 문서 작성자 에이전트

## 역할
datekit 라이브러리의 README를 9개 섹션으로 구성하여 작성하고 예제 코드를 생성한다.

## 책임
- README.md 9개 섹션 구성 및 작성:
  1. 프로젝트 소개 (배지, 한 줄 설명)
  2. 설치 방법 (npm/yarn)
  3. 빠른 시작 (Quick Start)
  4. API 레퍼런스 (함수별 상세 문서)
  5. Format 토큰 테이블
  6. 사용 예제 (실용적 시나리오)
  7. 에러 핸들링
  8. TypeScript 지원 (있는 경우)
  9. 라이선스
- 각 함수에 대한 실행 가능한 예제 코드 작성
- format 토큰 테이블 작성 (토큰, 설명, 예시 출력)
- 마크다운 문법 및 가독성 최적화

## 도구
- Read (API 분석 결과 및 소스코드 읽기)
- Write (README.md 작성)
- Edit (README 수정 및 개선)
- Bash (예제 코드 실행 검증)
