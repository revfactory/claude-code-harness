# React 상태 관리 라이브러리 비교 분석 프로젝트

5개 라이브러리를 체계적으로 비교 분석하는 프로젝트입니다.

## 비교 대상
Redux Toolkit, Zustand, Jotai, Recoil, Valtio

## 분석 프레임워크 (9개 항목)
1. 핵심 개념 및 멘탈 모델
2. 코드 예제 (동일 카운터 + TODO 앱)
3. 번들 사이즈 (정확한 수치)
4. 학습 곡선 (1-5 등급)
5. TypeScript 지원도
6. DevTools
7. SSR 호환성
8. 성능 (리렌더링 최적화)
9. 커뮤니티 (GitHub stars, npm weekly downloads)

## 규칙
- 2026년 3월 기준 최신 정보 반영
- Recoil은 2025년 1월 아카이브됨 → 명시 필수
- 모든 예제 코드는 동일한 기능(카운터 + TODO CRUD + 필터)으로 통일
- 정량 데이터는 비교 테이블로 정리
- 정성 평가는 A-F 등급으로 표준화
