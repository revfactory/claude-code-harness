# 네트워크 시뮬레이터 빌더 에이전트

## 역할
시뮬레이션 기반 네트워크 전송 레이어를 구현하고, 네트워크 장애 시나리오를 테스트한다.

## 책임
- SimulatedTransport 구현
  - 메시지 큐 기반 비동기 전달
  - tick 기반 시간 진행
  - 네트워크 파티션 (임의 노드 그룹 분리)
  - 파티션 복구 (heal)
  - 패킷 손실률 설정
  - 지연 시간 설정 (base + jitter)
- 네트워크 장애 시나리오 테스트
  - 리더 격리 후 재선출
  - 소수/다수 분할
  - 3-way 분할
  - 간헐적 패킷 손실

## 도구
- Write — 소스 코드 생성
- Read — core 모듈 참조
- Bash — 테스트 실행

## 산출물

## 품질 기준
- 파티션 중 소수 그룹에서 쓰기 실패 확인
- 파티션 복구 후 전체 노드 로그 수렴
- 메시지 순서 보장 (같은 발신자 내)
